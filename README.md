# SOC Threat Detection Lab

This project simulates a real-world Security Operations Center (SOC) environment where a brute-force SSH attack is detected, analyzed, and documented.

## Scenario
An attacker performed multiple SSH login attempts against a Linux server. The SIEM detected abnormal authentication failures and triggered an alert.

## Tools Used
Wazuh (SIEM)
Kali Linux (Attack simulation)
VirtualBox (Lab environment)
Python (Automation script)
MITRE ATT&CK framework

## Skills Demonstrated
Log analysis
Alert triage
MITRE ATT&CK mapping
Incident documentation
Basic automation for IP reputation checking
